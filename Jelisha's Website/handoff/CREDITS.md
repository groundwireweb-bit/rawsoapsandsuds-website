# Credits & Asset Licenses — Raw Soaps and Suds

## Icon artwork — OpenMoji (required attribution)

The scent-card icons (mango, cherry blossom, apple, herb/patchouli, and the
"savoring food" face for Eat Me) and the contact-page email/phone icons use
**OpenMoji** — the open-source emoji library.

- Source: https://openmoji.org
- License: **Creative Commons Attribution-ShareAlike 4.0 (CC BY-SA 4.0)** —
  https://creativecommons.org/licenses/by-sa/4.0/
- Files live in `assets/emoji/` (named by Unicode codepoint, e.g. `1F96D.svg`)
  and are referenced with `<img>` in `index.html`, `shop.html`, and `contact.html`.
  Each usage is marked in the HTML with a `<!-- Icon: OpenMoji ... CC BY-SA 4.0 -->`
  comment.

**What CC BY-SA 4.0 requires of you:**
1. **Attribution** — credit "OpenMoji" and link the license. This file plus the
   in-page HTML comments provide it. If you'd like the credit to be visible on the
   site itself (some prefer this), add a short line to the footer, e.g.
   *"Icons by OpenMoji (CC BY-SA 4.0)."*
2. **ShareAlike** — if you modify an OpenMoji SVG, the modified icon must be shared
   under the same CC BY-SA 4.0 license. The files here are used unmodified.

These replaced the earlier hand-drawn (AI-generated) inline SVG icons, at the
client's request to use open-source artwork instead of AI-generated artwork.

## Brand mark — original to this project

The bluebonnet "maker's stamp" (nav mark `assets/logo-mark.svg`, favicon
`assets/favicon.svg`, social image `assets/og-image.png`, and logo `assets/logo.png`)
is bespoke artwork created for Raw Soaps and Suds. It is not an emoji/stock asset
and carries no third-party license. There is no bluebonnet in OpenMoji, and the
brief called for a custom maker's badge, so this remains the site's own mark.

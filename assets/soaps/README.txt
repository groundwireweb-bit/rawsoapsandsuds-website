HOW TO ADD REAL SOAP PHOTOS
===========================

Drop a photo into THIS folder (assets/soaps/) using the exact file name below,
and it will appear on the Home and Shop pages automatically. No code editing.

Until a photo exists, the card shows a small placeholder icon instead.

File names (must match exactly, all lowercase, .jpg):

    mango-madness.jpg      -> Mango Madness
    cherry-blossom.jpg     -> Cherry Blossom
    apple-fantasy.jpg      -> Apple Fantasy
    perfect-patchouli.jpg  -> Perfect Patchouli
    eat-me.jpg             -> Eat Me

Tips:
- Roughly landscape works best (the frame is 4:3, e.g. 1200 x 900 px). Other
  shapes are fine too; the photo is centered and cropped to fit the frame.
- A JPG or PNG both work. Keep the name ending in ".jpg" as shown above.
- Keep files reasonably small (about 200-500 KB each) so pages load fast.
- To add a photo for a new scent later, name it <scent-slug>.jpg and update the
  matching card's <img src="assets/soaps/..."> in index.html and shop.html.
